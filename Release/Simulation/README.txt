README: Running Code
This readme file contains instructions on how to run the set of code.

Prerequisites
Before you start, please ensure that the following software is installed on your computer:
MATLAB
Any other dependencies required by the code


Running the Code
To run the code, follow these steps:
Open MATLAB

Invpend1_nonlin.slx is the Simulink file 

Simgraphsnoanimation.m can be run directly

To run Simwanimation.m, pendcart.m and drawpend.m must be open so that the animation of control can be displayed. 

If you encounter any issues while running the code, please check the error messages and refer to the documentation.
